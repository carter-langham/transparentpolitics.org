What is Transparent Politics?

This website's main purpose is to spread reliable information about sociopolitcal and economic issues in the United States. At the moment, Transparent Politics is going to be used to track anti-trans legislation, school shootings, sexual assault committed by church/government officials, and highlight the path of congress donations. The data presented on this website will have sources and be properly audited for accuracy unless otherwise stated. If you have any feedback (ideas, ​criticism, etc) please fill out the form below.
